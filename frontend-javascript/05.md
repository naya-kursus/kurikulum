### Latihan
Terdapat sebuah array sebagai berikut:
```js
['Jakarta', 'Surabaya', 'Bandung']
```

Urutkan sesuai karakter terbanyak dalam tiap elemen.

Contoh:
```
karakter: a
output: ['Jakarta', 'Surabaya', 'Bandung']

karakter: n
output: ['Bandung', 'Jakarta', 'Surabaya']
```
